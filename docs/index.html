Dotfiles PPA

Personal development environment meta-package for Debian/Ubuntu systems

üöÄ Quick Installation

Download the signing key:
sudo curl -fsSL https://dotfiles.chrisglass.dev/cglass-devenv-keyring.gpg -o /usr/share/keyrings/cglass-devenv-keyring.gpg
Add the repository:
echo "deb [signed-by=/usr/share/keyrings/cglass-devenv-keyring.gpg] https://dotfiles.chrisglass.dev/packages/debian/ stable main" | sudo tee /etc/apt/sources.list.d/chrisglass-dotfiles.list
Update and install:
sudo apt update && sudo apt install cglass-devenv
This PPA is designed for personal use and development environments. Review the included packages before installation.
